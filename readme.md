# 🎬 CineVerse

CineVerse is an AI-powered movie recommendation web application built with **Python** and **Django**. It helps users discover movies across multiple languages with personalized recommendations, advanced search, filtering, and an intuitive user interface.

---

## ✨ Features

- 🔐 User Registration & Login Authentication
- 🎥 Browse a large collection of movies
- 🤖 AI-based Movie Recommendations
- 🔍 Search movies by title
- 🌍 Filter movies by language
- ⭐ Filter by IMDb rating
- 📅 Filter by release year
- 📖 Detailed movie information
- 🖼️ Movie posters
- ❤️ Wishlist (In Progress)
- 📱 Responsive user interface

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/yourusername/CineVerse.git
```

### Navigate to the project

```bash
cd CineVerse
```

### Create a virtual environment

```bash
python -m venv .venv
```

### Activate the virtual environment

Windows

```bash
.venv\Scripts\activate
```

Linux/macOS

```bash
source .venv/bin/activate
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Apply migrations

```bash
python manage.py migrate
```

### Import Movie Dataset

```bash
python manage.py import_movie
```

### Run the server

```bash
python manage.py runserver
```

Open your browser:

```
http://127.0.0.1:8000/
```
